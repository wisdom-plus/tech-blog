# Style Conventions
- TypeScript with `strict: true`.
- React function components and async server components in the App Router.
- Formatting uses 2-space indentation, semicolons, and double quotes (see `biome.json`).
- Import alias `@/*` maps to `src/*`.
- Existing code uses simple named constants, camelCase variables/functions, and PascalCase React components.
- Biome organizes imports via assist actions.
